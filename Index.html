• git config --global user.name blabla - встановлює ім’я(blabla), яке можна ідентифікувати під час перегляду історії версій
  • git config --global user.email blabal@bla - встановлює імейл ( blabal@bla) який можна ідентифікувати з маркером в історії
  • git init - ініціює існуючу дирекиторію як git репозиторій
  • git status - показує змінені файли в робочій директорії, підготовлені до наступного коміту
  • git add - додає файли які виглядають як наступній коміт
  • git commit - m - комітить вміст на стейджі як новий комітний знімок
  • git log - показує всі коміти в історії певної гілки
  • git checkout - переключає на іншу гілку показує її в робочій директорії
  • git branch - показує перелік гілок. * з’являється біля активної гілки
  • git push - переносить локальні коміти гілки на гілку віддаленого репозиторія
  • git pull - збирає и мерджить коміти з віддаленої гілки
  • git merge - мерджить віддалену гілку в твою поточну гілку і оновлює інформацію
  • git clone - дістає весь репозиторій з хостинга через URL
  • .gitignore - файл, що містить список файлів , які ігноруватиме git
